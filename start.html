UNSCRAMBLE THE WORDS

RULES:

  1. Unscrammble each word within 60 seconds and get 5 points.
  2. Use Show Hint button to get a hint and guess the correct word within time limit.Get 3 points for each correct word if Show Hint button is used.
  3. Use Change Word button to go to the next word.
  4. Use Submit button to submit your answer. Multiple submissions allowed if Show Correct Answer and the Time Limit does not exceed.
  5. Use Show Correct Answer button to see the correct answer.No submission of the same word will be allowed if the Show Correct Answer button is already used for the word.
  6. Only Three Chances of failures will be allowed. If total number of failures( the number of times Show Correct Answer used +the number of times the Time limit exceeded)reaches three,the game ends there.
